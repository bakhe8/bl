# وثيقة قالب الخطاب — أولوية 4/5

## القالب الرسمي
- صياغة عربية ثابتة لخطاب تمديد الضمان البنكي.
- يدعم RTL بالكامل.

## Placeholders الأساسية
- اسم البنك الرسمي.
- اسم المورد الرسمي.
- رقم الضمان/العقد.
- المبلغ.
- تاريخ الانتهاء الحالي وتاريخ التمديد المطلوب (إن وجد).

## تنسيق التاريخ والمبلغ
- التاريخ: تنسيق عربي/هجري عند الحاجة، أو ميلادي بصيغة واضحة.
- المبلغ: قيمة رقمية مع عملة محددة (SAR مثلاً) وتنسيق آلاف ثابت.

## التنسيق الطباعي
- RTL، هوامش متساوية، خط عربي واضح.
- ترويسة/تذييل ثابتان بحسب الجهة.
- إعداد للطباعة من المتصفح مع خيار حفظ كـ PDF.

## الثبات
- أي تعديل في الصيغة أو placeholders يجب توثيقه هنا وتحديث القالب في التطبيق لضمان اتساق كل الخطابات.
