# وثيقة قالب الخطاب — أولوية 4/5

## القالب الرسمي
- صياغة عربية ثابتة لخطاب تمديد الضمان البنكي.
- يدعم RTL بالكامل ويستخدم HTML Template جاهز للطباعة.
- النص الرسمي يجب حفظه كوحدة واحدة (ترويسة، متن، تذييل) مع عدم تعديل الصياغة إلا بتوثيق مسبق.
- مواصفات الخط الطباعي: "Noto Naskh Arabic", "Segoe UI", serif. العنوان 18px، المتن 16px، line-height: 1.8.
- محاذاة RTL صريحة عبر `html, body { direction: rtl; unicode-bidi: bidi-override; }`.

## Placeholders الأساسية
- `{{bank_display}}`
- `{{supplier_display}}`
- `{{guarantee_number_display}}`
- `{{contract_number_display}}`
- `{{amount_display}}`
- `{{renewal_date_display}}`
- `{{sender_name}}`
- `{{sender_position}}`
- `{{department_name}}`
- `{{hospital_name}}`

## تنسيق التاريخ والمبلغ
- التاريخ: تنسيق عربي واضح بصيغة "01 يناير 2025م" (يومان رقمين، شهر عربي كامل، سنة رقمية، حرف م).
- المبلغ: قيمة رقمية بالأرقام العربية-الهندية مع فاصل آلاف عربي `٬` وخانتين عشريتين `٫` ثم كلمة "ريال" (مثال: **٩٩٩٬٩٩٩٫٩٩ ريال**). لا تحويل للمبلغ إلى نص في هذه النسخة.

## التنسيق الطباعي
- RTL، هوامش متساوية، خط عربي رسمي.
- ترويسة/تذييل ثابتان بحسب الجهة.
- إعداد للطباعة من المتصفح مع خيار حفظ كـ PDF.
- يوصى بخط افتراضي واضح (مثلاً "Noto Naskh Arabic", "Segoe UI", serif) بحجم العنوان 18px والمتن 16px ومسافة أسطر 1.8 مع هوامش 20mm.
- ثبّت ترميز الصفحة على UTF-8 لتفادي اختلاف الحروف في PDF.
- CSS طباعة لتقليل اختلاف المتصفحات:
  ```
  @media print {
    @page { size: A4; margin: 20mm; }
    body { -webkit-print-color-adjust: exact; print-color-adjust: exact; }
  }
  ```
- يمكن تثبيت WebFont داخل الصفحة إذا توفر لضمان توحيد الخط في PDF بين Chrome/Edge.

## الثبات
- أي تعديل في الصيغة أو placeholders يجب توثيقه هنا وتحديث القالب في التطبيق لضمان اتساق كل الخطابات.
- احتفظ بنسخة HTML في `templates/letter_template.html` مع تعليق يحوي رقم إصدار القالب وتاريخه.
- أضف معاينة ثابتة (نموذج بيانات) لاختبار المحاذاة بعد أي تغيير.

## النص الرسمي المعتمد (يُستخدم كما هو داخل القالب)
```
السادة / {{bank_display}}
المحترمين

السلام عليكم ورحمة الله وبركاته، وبعد،،

الموضوع: طلب تمديد الضمان البنكي رقم ({{guarantee_number_display}}) والعائد للعقد رقم ({{contract_number_display}})

إشارةً إلى الضمان البنكي النهائي الموضح أعلاه، والصادر منكم لصالح {{hospital_name}}، 
على حساب {{supplier_display}}، بمبلغ وقدره ({{amount_display}}) ريال سعودي.

نأمل منكم التكرم بالموافقة على تمديد سريان الضمان المذكور حتى تاريخ ({{renewal_date_display}})، 
مع تزويدنا بنسخة من التمديد متى ما تم ذلك.

علمًا بأنه في حال عدم تمكنكم من تمديد الضمان قبل انتهاء مدة سريانه، 
فيجب عليكم سداد قيمة الضمان المذكور لصالح المستشفى حسب الأنظمة المرعية.

شاكرين ومقدرين حسن تعاونكم،،،

وتفضلوا بقبول خالص تحياتنا وتقديرنا.

{{sender_name}}
{{sender_position}}
{{department_name}}
{{hospital_name}}
```
القالب موجود أيضاً بصيغة HTML في `templates/letter_template.html` مع CSS للطباعة ومحاذاة RTL.
