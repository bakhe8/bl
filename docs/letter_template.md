# وثيقة قالب الخطاب — أولوية 4/5

## القالب الرسمي
- صياغة عربية ثابتة لخطاب تمديد الضمان البنكي.
- يدعم RTL بالكامل ويستخدم HTML Template جاهز للطباعة.

## Placeholders الأساسية
- `{{bank_display}}`
- `{{supplier_display}}`
- `{{guarantee_number_display}}`
- `{{contract_number_display}}`
- `{{amount_display}}`
- `{{renewal_date_display}}`

## تنسيق التاريخ والمبلغ
- التاريخ: تنسيق عربي واضح (هجري/ميلادي عند الحاجة).
- المبلغ: قيمة رقمية مع عملة محددة (مثلاً SAR) وتنسيق آلاف ثابت.

## التنسيق الطباعي
- RTL، هوامش متساوية، خط عربي رسمي.
- ترويسة/تذييل ثابتان بحسب الجهة.
- إعداد للطباعة من المتصفح مع خيار حفظ كـ PDF.

## الثبات
- أي تعديل في الصيغة أو placeholders يجب توثيقه هنا وتحديث القالب في التطبيق لضمان اتساق كل الخطابات.
