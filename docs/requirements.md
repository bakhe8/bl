# المتطلبات

## المتطلبات الوظيفية
- رفع ملف Excel واحد يحتوي صفوف الضمانات.
- تحويل Excel إلى JSON داخلي موحّد.
- تنظيف القيم (أسماء بنوك/موردين، مبالغ، تواريخ) وتوحيدها بالصيغ الرسمية.
- استخدام قاموس ذكي يتعلم من اختيارات المستخدم ويُسجل الفروقات (variants) لحل الغموض لاحقاً.
- عرض السطور التي لم تُحل تلقائياً لقرار سريع من المستخدم.
- توليد خطاب رسمي بالعربية جاهز للطباعة عبر المتصفح (Print → Save as PDF).
- استخدام قالب خطاب موحد مع تثبيت اللغة العربية والاتجاه RTL.
- حفظ القواميس محلياً (LocalStorage + JSON) بلا خادم.

## المتطلبات غير الوظيفية
- العمل دون اتصال بالإنترنت ولا يعتمد على خوادم.
- سرعة عالية في الاستجابة (معالجة عشرات الصفوف خلال ثوانٍ).
- دقة وتناسق في الصياغة، مع الحد الأدنى للأخطاء البشرية.
- واجهة بسيطة وواضحة تدعم العربية بالكامل (نص واتجاه).
- سهولة الصيانة: لا تثبيت ولا إعدادات معقدة، وتشغيل مباشر بفتح ملف HTML.

## القيود والافتراضات
- نظام التشغيل المستهدف: Windows (أو أي نظام يدعم متصفح حديث).
- التشغيل مباشرة من المتصفح (Chrome/Edge) بفتح `index.html` محلياً.
- لا توجد قاعدة بيانات خارجية؛ التخزين محلي في LocalStorage ويمكن تصديره/استيراده كـ JSON.
- الاعتماد على مكتبة SheetJS صغيرة لقراءة Excel.

## نطاق العمل
- يغطي إصدار خطابات تمديد الضمانات البنكية فقط.
- يشمل إدارة القواميس لتوحيد أسماء البنوك والموردين وتعزيز التعلّم.
- لا يشمل تكاملات خارجية مع أنظمة أخرى أو قواعد بيانات حتى يتم تحديدها لاحقاً.
