# وثيقة الصيانة والتطوير المستقبلي — أولوية 2/5

## مهام شائعة
- **إضافة بنك/مورد جديد**: تحديث القاموس الرسمي وإضافة أي variants معروفة.
- **تعديل قالب الخطاب**: تحديث placeholders وتنسيق RTL والتاريخ/المبلغ مع توثيق التغيير في `docs/letter_template.md`.
- **تحديث الأعمدة**: تعديل column mapping وإعادة حفظه في `saved-config/`.
- **ترقية SheetJS**: اختبار استيراد/تصدير القواميس وتشغيل مسار كامل (Upload → Match → Letter).
- **نقل النظام لجهاز آخر**: تصدير القواميس وملفات الإعداد واستيرادها قبل الاستخدام.

## مبادئ
- أي تغيير في السلوك التطابقي يُسجّل في `docs/matching_algorithms.md`.
- أي تغيير في الحقول أو البيانات المدخلة يُحدّث في `docs/excel_processing.md`.
- حافظ على توافق إصدارات القاموس وملفات التكوين عند الترقية أو النقل.
