# الإعداد والتشغيل

## البيئة المقترحة
- نظام التشغيل: Windows (أو أي نظام يحوي متصفح حديث).
- المتصفح: Chrome أو Edge.
- الاتصال: غير مطلوب؛ التطبيق يعمل أوفلاين بالكامل.
- الخادم: لا يوجد (Serverless). التشغيل محلي بفتح ملف HTML.

## خطوات التشغيل المبدئية
1) استنسخ المستودع محلياً.
2) افتح ملف `index.html` في المتصفح (سيُضاف مع بناء الواجهة).
3) حمّل ملف Excel للضمانات عبر الواجهة.
4) عالج السطور الغامضة، ثم اطبع الخطاب بصيغة PDF من المتصفح (Print → Save as PDF).

## الاعتمادات
- SheetJS مدمجة لقراءة ملفات Excel.
- التخزين في LocalStorage للقاموس، مع خيار التصدير/الاستيراد بصيغة JSON.

## النسخ الاحتياطي والاستعادة
- وفّر خيار تصدير القاموس إلى ملف JSON دوريًا (خاصة عند العمل على أكثر من جهاز).
- للاستعادة، استورد ملف JSON لإعادة تدريب القاموس سريعاً.

## الدعم المستقبلي
- عند إضافة حزم/بناء (npm أو غيره) حدّث ملف CI في `.github/workflows/ci.yml` وخيارات التخزين/النسخ الاحتياطي هنا.
