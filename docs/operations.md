# التشغيل اليومي وإدارة القواميس

## سير العمل اليومي
1) افتح التطبيق المحلي.
2) حمّل ملف Excel للضمانات (ملف واحد في كل دفعة).
3) راجع السطور التي لم تُحل تلقائياً وحدد الصيغة الصحيحة لأسماء البنوك والموردين.
4) احفظ القرارات لتحديث القاموس تلقائياً.
5) أنشئ الخطاب واطبعه/صدّره PDF من المتصفح.

## إدارة القواميس
- كل اختيار جديد يُضاف إلى القاموس (LocalStorage) لتقليل الأسئلة لاحقاً.
- استخدم أسماء رسمية موحدة (عربية) للبنوك والموردين.
- وفّر أزرار تصدير/استيراد JSON لنسخ القاموس احتياطياً أو لمشاركته بين الأجهزة.

## معالجة الحالات الغامضة
- اعرض فقط الصفوف التي لم تُطابق تلقائياً لتسريع القرار.
- استخدم البحث بالاسم الجزئي أو المبلغ/التاريخ لتأكيد الهوية.
- عند تثبيت قرار جديد، يُخزَّن كـ variant لتكرار استخدامه.

## جودة الإخراج
- التزم بقالب الخطاب الموحد (RTL، لغة عربية رسمية).
- تحقّق من الحقول الرئيسية قبل الطباعة: اسم البنك، اسم المورد، المبلغ، تاريخ الانتهاء، رقم الضمان/العقد.
- شارك الخطاب عبر PDF محفوظ من المتصفح (Print → Save as PDF).

## صيانة بسيطة
- لا توجد خدمات خلفية؛ يكفي تحديث ملف HTML/JS عند الحاجة.
- احتفظ بملف القاموس المصدّر (JSON) كنسخة احتياطية دورية.
